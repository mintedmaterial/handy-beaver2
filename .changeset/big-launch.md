Initial release of The Handy Beaver app

Features:
- Custom domain handybeaver.co configured
- GitHub OAuth admin authentication
- Customer portal with magic link auth
- Admin dashboard (quotes, customers, messages, invoices, schedule)
- Admin API with 15+ endpoints for Lil Beaver agent
- Facebook page integration (posting, insights)
- Lead classification system for Facebook group monitoring
- Cloudflare Browser Rendering binding
- Discord webhook notifications
- Email handler for Cloudflare Email Routing
- D1 database with 14 tables
- R2 bucket for images

Tech stack:
- Hono on Cloudflare Workers
- D1 database
- R2 storage
- Browser Rendering
- TypeScript
